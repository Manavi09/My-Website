COVID-19 Chatbot

Overview

This project demonstrates how to build a basic chatbot capable of answering frequently asked questions about COVID-19. It leverages Natural Language Processing (NLP) techniques in Python to parse user queries, compare them against a predefined knowledge base, and return relevant, concise answers. By combining data cleaning, similarity scoring, and user‐friendly output, this chatbot offers a quick reference tool for COVID‐19 information.

Key Features

Data Preparation
Gathers and cleans a set of COVID‐19 FAQs, filtering out extraneous text so that each question‐and‐answer pair remains clear and consistent.
Applies tokenization and lemmatization to normalize the text data.

Similarity Scoring
Uses TF‐IDF or cosine similarity (depending on the method chosen in the code) to compare user queries with the existing FAQ knowledge base.
Selects the best‐matched answer for user‐typed queries, improving the chatbot’s relevance and accuracy.

User Interaction
Facilitates a simple, text‐based chat session where users can type natural language questions about symptoms, prevention measures, testing, and more.
Returns the most suitable, contextually relevant answer pulled from the knowledge base.

Easy Integration
Written in Python and presented in a Jupyter Notebook, allowing for straightforward deployment or further enhancement.
Readily extensible: developers can add more FAQs, incorporate updated health guidelines, or switch out NLP models for improved performance.

Technologies & Libraries

• Python for the main application logic and data processing
• NLTK / SpaCy (as shown in the code) for tokenization and lemmatization
• Scikit‐learn or similar libraries for TF‐IDF vectorization and similarity computations
• Jupyter Notebook environment for clear, iterative development

Outcome

This chatbot provides a user‐friendly interface to quickly retrieve COVID‐19 health information, showcasing how NLP techniques can transform a static FAQ list into an interactive, on‐demand knowledge system. It also illustrates core chatbot development concepts—from data cleaning and feature extraction to similarity scoring—that can be applied to broader conversational AI projects.
